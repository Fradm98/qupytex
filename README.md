# qupytex

qupytex is a Quantum Phase Transition Explorer Python package that enables researchers to characterize the complex landscape of Quantum Phase Transitions in Quantum Many-Body Systems.

## Installation

You can install qupytex either by cloning the repository or using pip.

### Clone the Repository

To clone the repository locally, use the following command:
